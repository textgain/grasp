# Grasp.py – language models

**Grasp** has optional, pre-trained language models for:

- language identification (`~lang.json.zip`)
- part-of-speech tagging (`*-pos.json`) 
- sentiment analysis (`*-pov.json`)
- word frequency (`*-doc.json`)
